# Dental-Perfect Admin

#### 项目介绍
[Dental-Perfect](http://www.dental-perfect.com/) 后管系统前端

#### 架构核心
* ESLint - 代码格式化工具
* roadhog - 项目启动打包工具
* babel - ES6编译工具
* REST API - API接口标准
* dva - redux框架
* ant-design - UI组件
* ant-design-pro - Ant-Design最佳实践脚手架
* parse-server - 后端数据服务
* parse-dashboard - 后端数据管理

#### 使用说明
* git clone https://
* npm start
